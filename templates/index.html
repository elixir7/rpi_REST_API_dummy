Welcome to the CASE WEB API for Ultimaker 3 printers

You can only send GET requests with this API.

READ

This is an API to get data from the printers, only /api/v1/printer has dummy data (JSON)

Printer

URI: /api/v1/printer will give you everything that has to do with the printer

Print_job

URI: /api/v1/print_job will give you everything that has to do the current print_job

History

URI: /api/v1/history will give you the last 50 print jobs and info about them

System

URI: /api/v1/system will give you data about the system and network

Camera

URI: /api/v1/camera will give you the adress to the ultimaker local wifi camera ip, there you can see the video feed live in high frame rate ;)
